Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents visible content shown to the user
  • Meta elements are always inside of the head element

CSS

  • Adds style to the webpage to make it more visually appearing to the user
  • A declaration contains a property and its assigned value
  • The link element can be used to link a CSS file to the main branch of code
  • A margin indicates how much space we want around an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates new branch and switches to it

JavaScript

  • Adds logic and functionality to the webpage
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Functions can be defined and then called upon in order to run the code inside the function
  • A for loop executes a single block of code repeatedly until all items in an array are ran
